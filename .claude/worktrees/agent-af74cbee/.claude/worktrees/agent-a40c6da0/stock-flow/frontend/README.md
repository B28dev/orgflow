# StockFlow Frontend

Base visual da Etapa 1 do StockFlow.

## Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- React Router
- Lucide React

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`

## Estrutura
- `src/app`: app shell, layout e router
- `src/components`: primitives, feedback e blocos visuais
- `src/screens`: telas base navegáveis
- `src/domain`: adapter da definição compartilhada e seletores
- `src/mocks`: dados mockados para a casca visual
- `src/styles`: tokens e estilos globais

## Fonte de verdade funcional
- `../docs/v1-foundation.md`
- `../shared/product-definition.json`
